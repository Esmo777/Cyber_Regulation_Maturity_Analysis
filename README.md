# Cyber_Regulation_Maturity_Analysis
project analyzing global cyber regulatory maturity across 194 countries using Python, and Power BI to support data-driven policy decisions.

📌 Overview

This project analyzes national cyber regulatory maturity using a macro-institutional dataset covering 194 countries.
The objectives are to:

- Assess regulatory maturity levels
- Identify strategic gaps
- Compare regions and income groups
- Support data-driven policy decisions

The project showcases a full analytics pipeline, from raw data exploration to executive reporting and interactive dashboards.

📂 Dataset

Scope
- 194 countries
- Regulatory maturity score (2024)
- Progress and priority indicators
- Geographic region
- World Bank income group

Data Characteristics
- CSV, structured tabular format
- Mixed data types (categorical & numerical)
- no missing values
- Multi-year indicators aggregated for 2024 analysis

🛠 Tools Used

- Python (Jupyter Notebook) — Data loading, EDA, cleaning
- Power BI — Interactive dashboards
- Gamma — Executive presentation

🔎 Project Workflow
1️⃣ Data Loading

- Load dataset in Python
- Inspect structure, data types, and missing values
- Validate dataset dimensions

2️⃣ Exploratory Data Analysis (EDA)

- Descriptive statistics
- Distribution of maturity scores
- Regional and income group comparisons

Goal: Understand global maturity patterns before deeper analysis.

3️⃣ Data Cleaning & Preparation

- Standardize column names
- Handle missing values
- Ensure numeric consistency
- Remove duplicates
- Create derived indicators (progress, gaps)
- Cleaned data is exported for SQL analysis.

4️⃣ Power BI Dashboard

The dashboard includes:

- Global maturity KPI
- Regional performance comparison
- Income group segmentation
- Strategic gap visualization
- Country-level drill-down

Designed for decision-makers to quickly identify priority areas and reform needs.

 5️⃣ Reporting & Presentation

Two executive deliverables:

📄 PDF Report — Decision-oriented insights

📊 PowerPoint (Gamma) — Clear, visual storytelling

Both focus on translating analytics into actionable policy insights.

📈 Key Insights

- Global regulatory maturity is moderate but highly uneven
- Strong link between income level and regulatory sophistication
- Low-income regions face structural capacity gaps
- Progress is incremental, increasing modernization risks
- Strategic gaps matter more than simple country rankings

🚀 How to Run the Project

- Run notebooks in /notebooks.
- Power BI
- Open .pbix file
- Refresh data
- Explore dashboards
